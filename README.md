# 🩺 Medical Dictionary Web App

A simple **Medical Dictionary Web Application** that allows users to search for medical terms and instantly view their meanings.

This project uses the **Wikipedia API** to fetch medical definitions and also includes a **Text-to-Speech feature** to read the meaning aloud.

---

## 🚀 Features

* 🔍 Search medical terms instantly
* 📖 Fetch definitions using Wikipedia API
* 🔊 Listen to the meaning with Text-to-Speech
* ⚡ Fast and lightweight
* 🎨 Simple and clean user interface

---

## 🛠️ Technologies Used

* **HTML** – Page structure
* **CSS** – Styling and layout
* **JavaScript** – Search logic and API integration
* **Wikipedia API** – Fetch medical term definitions
* **Web Speech API** – Text-to-Speech functionality

## ⚙️ How It Works

1. User enters a **medical term**.
2. The application sends a request to the **Wikipedia API**.
3. The API returns a short **summary of the term**.
4. The result is displayed on the page.
5. Users can click the **🔊 Listen button** to hear the meaning.
